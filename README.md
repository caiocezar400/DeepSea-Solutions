# DeepSea-Solutions
Repositório para os códigos da entrega final para a matéria de Problemas V e Dinâmica de sistemas robóticos.




Códigos feitos no programa Google Colab utilizando a linguagem Python. Para acessar o código, em todos os exemplos que foi utilizado Colab estará disponível um link direto para a utilização do mesmo.
Todos foram feito de forma generalizada, permitindo a entrada das variáves, então é importante estar com os parâmetros em mãos antes de iniciar os testes de cada.


Os códigos de Cinemática Direta, Cinemática Inversa e Jacobiano podem ser rodados tranquilamente utilizando a ferramenta "Executar tudo" dentro do software.
O código de trajetória é necessária inicialmente rodar exclusivamente a parte de importação da biblioteca, que se encontra na primeira divisão do código. Atente-se que pode levar um tempo até que a 
primeira parte seja concluída. Depois de feito esse processo basta rodar a segunda parte de código e digitar os parâmetros.
